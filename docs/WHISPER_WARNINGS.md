# ℹ️ Предупреждения Whisper

## Предупреждение: "FP16 is not supported on CPU; using FP32 instead"

### Что это значит?

Это **информационное предупреждение**, не ошибка. Whisper сообщает, что:
- FP16 (полуточность) не поддерживается на CPU
- Автоматически используется FP32 (полная точность)

### Это нормально?

✅ **Да, это абсолютно нормально!**
- На CPU всегда используется FP32
- Это не влияет на качество распознавания
- Это немного медленнее, чем FP16 на GPU, но работает корректно

### Можно ли отключить предупреждение?

✅ **Да, предупреждение уже отключено в коде!**

В `backend/app/main.py` добавлен фильтр:
```python
warnings.filterwarnings("ignore", message="FP16 is not supported on CPU; using FP32 instead")
```

После перезапуска сервера предупреждение не будет показываться.

### Другие предупреждения Whisper

**Предупреждение о torchaudio:**
```
torchaudio._backend.list_audio_backends has been deprecated
```

Это также информационное предупреждение от библиотеки `torchaudio`. Не влияет на работу. Библиотека `pyannote.audio` использует устаревший API, но всё работает корректно.
